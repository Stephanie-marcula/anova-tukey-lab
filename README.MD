# ANOVA e Tukey Lab

Aplicativo em Streamlit para análise estatística de ensaios experimentais.

## Funcionalidades

- Upload de planilhas Excel ou CSV
- ANOVA de uma via
- Teste post hoc de Tukey
- Teste de Shapiro-Wilk
- Teste de Levene
- Gráfico dos intervalos de confiança de 95%
- Exportação em Excel, PNG e PDF

## Como rodar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
